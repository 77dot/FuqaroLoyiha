Направления деятельности компании

Проектная организация ООО «FUQAROLOYIHA» осуществляет следующие виды работ:

  • Разработка проектно-сметной документации по стадиям: «Технико-экономическое обоснование» (ТЭО), и «Технико-экономический расчет» (ТЭР);
  • Разработка проектно-сметной документации по стадиям: «Рабочий проект» (РП) и «Рабочая документация» (РД);
  • Разработка планировочных решений архитектурно-строительной части;
  • Разработка дизайн-проектов экстерьеров и интерьеров;
  • Разработка проектов по электроснабжению;
  • Разработка проектов по сетям водопровода и канализации;
  • Разработка проектов по отоплению и вентиляции-кондиционированию;
  • Разработка проектов по слаботочным сетям (охранной и пожарной сигнализации);
  • Разработка проектов архитектурно-планировочной организации территории (АПОТ);
  • Разработка проектов организации строительства (ПОС) и проектов производства работ (ППР);
  • Разработка проектов энергоэффективности;
  • Спецификации по комплектации отделочных материалов;
  • Составление и разработка сметной документации;
  • Проведение авторского надзора за строительством объекта.
